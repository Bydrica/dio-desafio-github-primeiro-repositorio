Sha1

* Cria 40 dígitos únicos.
* É uma forma curta de representar um arquivo.

Sha1 e Objetos Fundamentais

* No Git os arquivos são armazenados em objetos chamados blob.
* As tees armazenam blobs e apontam para outra árvores.
* Commit é o objeto que junta tudo.
(Todos possuem Sha1)

Chave SSH

* É uma forma de estabelecer uma conexão segura e encripitada enter duas máquinas.

Iniciar o Git e Criar um Commit

* Iniciar o Git: /.git init
* Iniciar o Versionamento: /.git add
* Criar um Commit: /.git commit
